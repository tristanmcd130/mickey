# mickey

A statically-typed imperative programming language that compiles to [MIC-1](https://en.wikipedia.org/wiki/MIC-1) assembly.

# Features

- Most arithmetic you would expect, including addition, subtraction, multiplication, division, modulo (these last 3 are in software and don't require a PROM that supports them as instructions)
- Comparison and logical functions
- Functions
- Stack-allocated local variables
- Global variables
- Static type checking. Types include integers, booleans, void, and pointers to any of those
- A memory allocator
- Ability to import other files
- Inline assembly
